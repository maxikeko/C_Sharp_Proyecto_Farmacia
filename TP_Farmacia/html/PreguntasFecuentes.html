Debes tener el usuario como administrador para ver todas las opciones. Si no ves todas los botones en el programa desbloqueados quiere decir que tienes un usuario limitado por el administrador Informale a un superior, para que te actualize tu usuario para tener todas las funciones disponibles.

Solo el administrador puede dar de altas los lotes de Remedios. Una vez tengas la usuario correspondiente, solo tendras que acceder a "Menu" y elegir la opcion "Compra a Proveedores", tendras que rellenar los datos del proveedor y del lote para efectuar la alta.

Para manejar cuentas corrientes en la farmacia. Se ofrece al cliente la posibilidad de crearse una cuenta con nosotros para tener un limite de saldo para pagar a final de mes en caso de no contar con efectivo.

Desde el "menu"- "Crear nueva venta" Apartir de aqui, tendras que seleccionar el cliente, buscar el producto en el stock y agregarlo al "carrito de compras", comprobaras si el cliente adeuda, y le daras a terminar Venta, podrias ver un listado de ventas en la seccion Menu-Ventas- "ver ventas".
